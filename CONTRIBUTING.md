# Contributing

Thanks for your interest in improving this project.

## Ways to help

- Report bugs with reproducible steps
- Suggest practical feature improvements
- Fix parsing/crawling edge cases
- Improve GUI usability and filtering
- Improve docs and examples

## Contribution guidelines

- Keep changes focused and small when possible.
- Include clear notes on what changed and why.
- Keep behavior backward compatible unless clearly documented.
- For bug fixes, include a short before/after description.

## Scope notes

- External links should be reported, not crawled.
- Keep crawler behavior transparent and easy to audit.
